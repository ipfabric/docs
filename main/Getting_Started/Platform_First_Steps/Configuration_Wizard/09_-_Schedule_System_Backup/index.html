09 - Schedule System Backup

Backup

Use IP Fabric backup to protect your important data.

There are two types of backup:

  • Local backup
  • Remote backup (FTP, sFTP)

Tip

Backups are encrypted with osadmin user password configured during thefirst boot wizard. When you loseosadmin user password, backups are also lost!
