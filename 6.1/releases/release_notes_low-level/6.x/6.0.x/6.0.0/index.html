    LLRN 6.0.0

    These are low-level release notes for IP Fabric release 6.0.0. Please note, that this page contains very low-level information about the actual release, which can lead to false conclusions if you dont have access to the tickets. On the other hand, it can provide valuable information, if you are looking for a particular detail. This release of IP Fabric contains a total of 580 issues.
