#### Path Lookup

Path Lookup has been extended to support IPv6 for both Unicast and Host to Gateway scenarios. The following technologies are now supported for IPv6 path analysis: routing, security evaluation, 6in4, GRE, and IPsec. NAT and PBR are not yet supported for IPv6 lookups.

![release_7.9_ipv6_pathlookup.webp](../../images/miscellaneous/release_7.9_ipv6_pathlookup.webp)

Path Lookup has been extended to support Layer 3 routing between cloud-only devices with no interface names.

![release_7.9_cloud_routing_without_interface_in_e2e.webp](../../images/miscellaneous/release_7.9_cloud_routing_without_interface_in_e2e.webp)



#### Vendor Support

- **Check Point Gaia** -- Improved NTP data collection by supporting the  extended `ntpq` command, providing detailed NTP source information. For more details, see [Extend command to get better NTP results](../../support/known_issues/Vendors/checkpoint.md#extend-command-to-get-better-ntp-results).
