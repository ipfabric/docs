# Subnets

This table provides a basic inventory of subnets in public clouds, including their ID, name, and IPv4 and IPv6 address ranges.

![Technology table showing Cloud subnets inventory](../../../images/technology/IP_Fabric_GUI-technology_tables-cloud_subnets.webp)

To get detailed vendor specific information about cloud subnets, navigate to **Technology --> Cloud --> Vendors --> AWS/Azure/GCP --> Subnets**.
